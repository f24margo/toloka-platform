# 🌱 Toloka Platform

Community bulletin board for Ukrainian villages.

## About

Toloka is a simple bulletin board for small Ukrainian communities.
Residents can post announcements, sell goods, offer services and share news.

## Features

- 📋 Announcements board with categories
- 📷 Photo upload (up to 3 photos per post)
- 🖼️ Photo slider on cards
- 🔍 Filter by category
- 👤 Only registered members can post
- ☁️ Cloudinary photo storage

## Categories

- 🥕 Продам (For sale)
- 🎁 Віддам даром (Free)
- 🔧 Послуги (Services)
- 📰 Новини/Події (News/Events)
- 💬 Інше (Other)

## Tech Stack

- **Wix** + **Velo** (JavaScript framework)
- **Wix CMS** (Announcements collection)
- **Cloudinary** (Photo storage)
- **HTML/CSS/JS** (Frontend iframe)
